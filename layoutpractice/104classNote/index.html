Path (環境安裝）

Mac安裝

環境安裝為重要的一環 , 把工具與環境安置好很重要

  • install homebrew -> 使用終端機安裝東西時會用到
  • donload and install item2
  • install zsh -> 美化終端機 , 安裝與設定完成後重啟終端機 , 使用指令 "chsh -s /bin/zsh" 即可套用(會發現字體會變成亂碼）
  • vi ~/zshsrc -> 進入 zsh 的設定檔案 , 如果有安裝 node.js etc.. 工具 , 都要從這裡印入到 zsh
  • 解決亂碼（字體問題）： brew tap caskroom/fonts -> brew cake install font-saucecodepro-nerd-font(超多字體可以選擇） -> command line :brew search nerd
  • 解決亂碼（字體問題）: item2 -> preformance -> text -> 選擇與切換字體
  • 樣式 : 進入連結下載樣式(color paste)
  • 切換樣式 : Open itemr2 > command + i > color > import > schemes floder > choose and import

如果 Mac toolbars 不見

進入活動監視器 => 磁碟 => 找到TouchBarServer => 選擇後左上角 Ｘ 強制結束
